# snapshots
Maven repository for personal snapshot artifacts

To use it, add a repository section in your pom.xml:

    <repositories>
        <repository>
            <id>atao60-repo</id>
            <name>atao60's Maven repository - snapshots</name>
            <url>https://github.com/atao60/snapshots/raw/master/</url>
        </repository>
    </repositories>
